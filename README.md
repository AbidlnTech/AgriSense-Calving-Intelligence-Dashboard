# AgriSense-Calving-Intelligence-Dashboard
AgriSense is a Power BI-based livestock intelligence dashboard that analyzes cow behavioral data, activity patterns, rumination trends, and calving indicators to support data-driven dairy farm management and early calving prediction.

AgriSense — Calving Intelligence Dashboard
Overview

AgriSense is a Power BI dashboard developed to analyze livestock behavioral data and identify patterns related to calving events. The dashboard provides insights into motion, rumination, standing hours, lying hours, and calving history to support data-driven dairy farm management.

Tools Used
Power BI
Power Query
DAX
Excel/CSV
Features
Executive Overview Dashboard
Behavioral Trend Analysis
Calving Prediction Analytics
Individual Cow Profile
Interactive Filters & KPIs
Key Insights
Activity levels increase near calving events.
Rumination patterns change before calving.
Standing and lying behavior provide useful indicators.
Behavioral trends help support early calving detection.
